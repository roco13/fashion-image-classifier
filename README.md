# Fashion Image Classifier (Client-Side)

A browser-based clothing classifier built with:

- React + TypeScript
- TensorFlow.js
- MobileNet v2
- Custom retail interpretation layer

## Features

- Image upload
- Client-side inference
- Retail category mapping
- Confidence visualization
- Model transparency

## Model

Uses MobileNet v2 pretrained on ImageNet.
Includes custom logic to map predictions into retail categories.
